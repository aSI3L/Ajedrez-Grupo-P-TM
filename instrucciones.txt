Instrucciones a seguir para pode ejecutar el Ajedrez y jugar:

1- Descargar el repositorio de GitHub. Link del mismo --> https://github.com/aSI3L/Ajedrez-Grupo-P-TM.git
2- Instalar en el sistema por consola/terminal: Python y dependencias (se describen a continuación los comandos)
  sudo apt install python3.9
  sudo apt install python3-pip
  sudo apt install python-tk
  sudo pip install pygame
3- Hacer que el sistema utilice el python 3.9 por defecto
  sudo update-alternatives --install /usr/bin/python3 python3 /usr/bin/python3.9 1
4- Por ultimo, correr el programa desde la consola/terminal
  Desde la terminal dirigirse al directorio donde estan los archivos .py
  y con el comando python3 interfazEjecutable.py se iniciará el juego
